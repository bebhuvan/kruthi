# Security Policy

## Reporting a Vulnerability

If you discover a security issue, please report it privately via GitHub Security Advisories:

- https://github.com/bebhuvan/kruthi/security/advisories

Include:
- A description of the issue and impact
- Steps to reproduce
- Any relevant logs or screenshots

We aim to acknowledge reports within 72 hours.

## Supported Versions

We support the latest release on the `main` branch. If you are using an older
version, please upgrade and verify the issue still reproduces.
