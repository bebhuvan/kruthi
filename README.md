# Kruthi

Kruthi is a trust-first EPUB reader with quoted answers, highlights, and local-first AI analysis. It runs on web, desktop, and mobile.

## Highlights

- Quote-first answers with citations back to the book.
- Local-first storage for books, highlights, and reading state.
- One Svelte codebase deployed to web, Tauri desktop, and Capacitor mobile.

## Quick Start

```bash
npm install
npm run dev
```

## Builds

```bash
npm run build
npm run build:tauri
npm run build:capacitor
```

## Desktop Downloads

Desktop installers are published on GitHub Releases after a tagged release:

```bash
git tag v0.1.0
git push origin v0.1.0
```

Releases: https://github.com/bebhuvan/kruthi/releases
